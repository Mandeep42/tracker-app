# Vue 3 + TypeScript + Vite

This project is developed based on Vue 3, Vuetify, Vuex and TypeScript in Vite. For icons, I have used "Font Awesome" and "mdi" library. This project is developed based on the Vue 3 composition api.

This project is responsible to create/resolve issue by registered users.

## Recommended IDE

- [VS Code](https://code.visualstudio.com/)

## Setup Project

Run `yarn && yarn run dev` from VS Code's command palette
